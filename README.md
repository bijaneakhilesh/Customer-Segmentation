# Customer Segmentation using K-Means

##  Project Overview

This project segments customers based on their income and spending behavior using K-Means clustering.

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

##  Steps Performed

* Data Cleaning
* Feature Selection
* Data Normalization
* K-Means Clustering
* Visualization
* Silhouette Score Evaluation

##  Results

Customers were grouped into 5 segments:

* High-value customers
* Potential customers
* Budget shoppers
* Inactive customers
* Regular customers

##  Insights

This segmentation helps businesses target customers with personalized marketing strategies.

##  Files Included

* Dataset (Mall_Customers.csv)
* Output file (customer_segmentation_output.csv)
* Jupyter Notebook (main.ipynb)
